Project Overview

For this assignment, I created a small real-time application that demonstrates concepts like task scheduling, timing guarantees, and predictable system behavior. Even though this is a lightweight version of a full project, it still highlights the core ideas behind designing software that responds within specific time limits.

How It Works

The program uses multiple tasks that run at different time intervals. A key idea in real-time systems is ensuring that high-priority tasks always run when needed, even when the processor is busy. The application also demonstrates deterministic timing, meaning the system responds the same way every time under the same conditions. This small demo acts as a foundation for larger real-time applications where timing, safety, and predictability matter.

About This Site

This webpage was created as part of the assignment to learn how to host a project using GitHub Pages. The steps included creating a repository, enabling GitHub Pages, writing a simple HTML file, and embedding a public YouTube video using an iframe.

In a real project, this website could expand into a full documentation page with detailed explanations, code walkthroughs, diagrams, and downloadable files.
